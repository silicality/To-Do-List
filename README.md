# To-Do-List
This repository contains a basic To-Do List application that is intentionally kept simple and minimal.
The goal of this project is not just to build a to-do app, but to create a real-world, challenge-based open-source project where contributors can gradually upgrade a basic implementation into a complete, production-ready application.
